# 🚀 Client Management System

A modern and responsive **Client Management System (CMS)** developed with **PHP, MySQL and Bootstrap**.

Manage clients, services, invoices and sales reports through a clean admin dashboard.

---

## ✨ Features

✅ Client Management  
✅ Service Management  
✅ Invoice Generation  
✅ Invoice Search System  
✅ Sales Reports  
✅ Authentication System  
✅ Password Recovery  
✅ Responsive Admin Dashboard  
✅ About Us Page Management  

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| PHP | Backend Development |
| MySQL | Database |
| Bootstrap | Responsive UI |
| HTML5 | Structure |
| CSS3 | Styling |
| JavaScript | Frontend Logic |
| jQuery | Dynamic Features |
